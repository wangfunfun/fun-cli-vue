### 使用方式

1. 下载代码仓库

```sh
git clone https://github.com/wangqf19/cli-vue3-vite2-ts.git
```

2. 安装npm依赖包
   
```sh
npm install
```

1. 运行开发环境

```sh
npm run dev
```

4. 构建生产环境

```sh
npm run build
```

1. 其他内置命令

本地预览生产环境<br/>
```sh
npm run preview
```

Prettier 代码格式化<br/>
```sh
npm run prettier
```

Eslint 检查代码格式<br/>
```sh
npm run lint
```

运行 Jest 单元测试<br/>
```sh
npm run test
```
